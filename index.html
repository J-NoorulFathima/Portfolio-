01 · about

Currently mid-stack-trace on a career pivot.

I'm Noorul, a developer running a deliberately intense self-study program across Java, Data Structures & Algorithms, SQL, and web development — treating interview prep less like memorization and more like an engineering problem worth building tools for.

That instinct is what produced the DSA Pattern Debugger: instead of reading about the two-pointer technique, you step through it, line by line, watching variables update in real time. It's grown into a 35+ pattern engine with flashcards, roadmaps, and full execution visualizations — all in one dependency-free HTML file, by choice.

Alongside the study track, I take on freelance AI-focused client work and run content strategy for a small business on the side. Every project I ship, I ship as a single self-contained file — no build step, no framework overhead, just something that works the moment you open it.
